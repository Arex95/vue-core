# Lab - Experimentación y Desarrollo

Esta carpeta contiene documentos y código relacionado con el desarrollo de nuevas funcionalidades.

## Archivos

- **`mvp-requirements.md`** - Requerimientos detallados para el MVP del generador de código desde OpenAPI
- **`generator-implementation-ideas.md`** - Diferentes enfoques de implementación (funcional, OOP, pipeline, templates)
- **`generator-oop-design.md`** - Diseño completo orientado a objetos con clases y estructura detallada

## Propósito

Esta carpeta es un espacio de trabajo para:
- Documentar requerimientos antes de implementar
- Experimentar con prototipos
- Planificar nuevas funcionalidades
